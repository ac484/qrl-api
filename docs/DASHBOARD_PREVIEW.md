## 顏色方案

- **QRL 卡片**: 藍紫色頂部邊框 (#667eea)
- **USDT 卡片**: 綠色頂部邊框 (#26de81)
- **價格卡片**: 紅色頂部邊框 (#fc5c65)
- **背景**: 漸層紫色 (#667eea → #764ba2)
- **卡片**: 白色背景，圓角，陰影效果

## 互動功能

1. **帳戶切換**
   - 點擊下拉選單切換主帳戶/子帳戶
   - 切換後自動重新載入數據

2. **自動刷新**
   - 每 30 秒自動更新所有數據
   - 無需手動刷新頁面

3. **手動刷新**
   - 點擊「重新整理」按鈕立即更新

4. **響應式設計**
   - 桌面: 3 欄佈局
   - 平板: 2 欄佈局
   - 手機: 1 欄堆疊

## 數據來源

所有數據通過以下 API 端點獲取：
- `/account/balance` - QRL & USDT 餘額
- `/market/ticker/QRLUSDT` - 價格和 24h 變化
- `/status` - 機器人狀態和交易統計
- `/account/sub-accounts` - 子帳戶列表

## 啟動步驟

1. 確保 FastAPI 服務運行
2. 訪問 http://localhost:8080/dashboard
3. 如果看到 API 錯誤，請配置 MEXC API 密鑰
4. 享受視覺化的交易監控！
